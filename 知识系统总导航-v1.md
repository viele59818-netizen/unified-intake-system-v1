# 知识系统总导航 v1

## 说明

这是一套面向“输入 - 思考 - 协作 - 执行 - 复盘”的轻量知识工作流。

当前版本目标：

- 先跑通
- 先能用
- 先形成统一入口

适用场景：

- 产品问题分析
- 售后问题分析
- 版本评估
- 客户反馈整理
- 内部知识沉淀

## 一、总结构

这套系统分成 5 层：

1. 输入层
2. 思考层
3. 协作层
4. 执行层
5. 复盘层

一句话对应关系：

- 输入层：收集事实
- 思考层：形成判断
- 协作层：共享结论
- 执行层：推进动作
- 复盘层：沉淀经验

## 九、推荐运行方式

1. 新问题进来，先用“快速采集卡片”记下来。
2. 需要系统整理时，转入“输入层模板”。
3. 让 AI 按提示词模板做整理和初步分析。
4. 把高价值内容沉淀进 Obsidian 主笔记。
5. 抽一版可共享内容到飞书文档。
6. 把行动项拆到滴答清单。
7. 最终结果回写 Obsidian，形成复盘。

## 十、后续迭代方向

## 十一、当前结论

当前这套 v1 已经可以实际使用，不需要再等“设计完全”才开始。

更好的方式是：

- 先用它跑真实案例
- 再根据卡点迭代 v2

这样系统会更贴近你的真实工作方式。
