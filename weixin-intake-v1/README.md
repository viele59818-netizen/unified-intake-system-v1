# 微信收集页原型 v1

这是一个最小可运行的收集页原型，用来验证下面这条链路是否顺手：

`手机端输入 -> 收集页 -> 输入池 -> 后续专题分析`

## 当前能力

- 填写专题名、标题、来源类型
- 粘贴原始内容和链接
- 选择图片和视频文件
- 将记录保存到浏览器本地 `localStorage`
- 查看提交记录
- 导出 JSON 或 Markdown

## 适合当前阶段的原因

- 不依赖后端也能先跑流程
- 适合先验证字段是否够用、操作是否顺手
- 后续可平滑替换成微信云开发 / OpenCloud 后端

## 本地打开方式

直接双击打开 [index.html](./index.html) 即可。

如果想用本地静态服务，也可以在当前目录运行：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```
